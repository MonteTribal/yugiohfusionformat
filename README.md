# yugiohfusionformat
generates a YGOPRODECK.com decklist based on fusions. Decks are not good, but meant to be played as limited against other decks generated by this tool. Decks are typically >60 cards, with the intent to trim down to something playable. Master Duel rarities not taken into account.
Please credit if used in a video or future project.
